> 구현해야 할 주요 기능들을 체크하고 구체화하는 단계입니다. 필요한 항목에 체크(`[x]`)해 주세요.

  * [ ] 일반 이메일 가입/로그인
  * [ ] 간편 소셜 로그인 ( [ ] 카카오 / [ ] 네이버 / [ ] 구글 / [ ] 애플 / [ ] 페이스북 )

  * [ ] 생체 인증 (Face ID, 지문 인식)
* **사용자 계정 및 프로필:**
  * [ ] 프로필 사진 및 닉네임 설정
  * [ ] 알림 설정 (푸시 알림 ON/OFF)
  * [ ] 회원 탈퇴 및 데이터 삭제 기능
* **커뮤니티 및 소통 기능:**

  * [ ] 댓글 / 대댓글 기능
  * [ ] 좋아요 / 북마크(스크랩) 기능
  * [ ] SNS 공유하기 (카카오톡, 링크 복사 등)
  * [ ] 1:1 채팅 또는 실시간 오픈 채팅
* **결제 및 커머스 기능:**
  * [ ] 일반 웹 결제/PG사 연동 (신용카드, 계좌이체 등)
  * [ ] 간편결제 연동 ( [ ] 네이버페이 / [ ] 카카오페이 / [ ] 토스페이 )
  * [ ] 인앱 결제 (App Store / Google Play 스토어 자체 결제 - 디지털 콘텐츠 필수)
* **위치 기반 서비스 (LBS):**
  * [ ] GPS 기반 사용자 현재 위치 파악
  * [ ] 지도 표시 및 마커 렌더링 (구글 지도, 네이버 지도, 카카오 지도 등)
  * [ ] 내 주변 장소 검색 및 거리 계산
* **디바이스 기능 제어:**
  * [ ] 카메라 및 사진 앨범 접근 (이미지 업로드용)
  * [ ] 푸시 알림 수신 (마케팅 및 알림용)
  * [ ] 오디오/마이크 접근 (음성 녹음 등)
  * [ ] 블루투스(BLE) 연동 (외부 기기 연결 필요 시)
